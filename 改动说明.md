## 三、管理员权限

### 3.1 管理员口令

`src/config.js` 第 17 行：

```javascript
isAdmin: "LELEO_SUPER_SECRET",
```

### 3.2 进入管理员模式

浏览器访问一次（只需一次，localStorage 记住）：

```
http://你的域名/?admin=LELEO_SUPER_SECRET
```

### 3.3 行为规则

| 角色 | 设置齿轮 | 壁纸/主题色 | 音乐 |
|---|---|---|---|
| 普通访客 | 完全不渲染（DOM 层面不存在） | 强制用 config.js 写死的值 | 可切换 |
| 管理员 | 可打开 | 可临时切换 + 🔒 发布为全局 | 可切换 |

### 3.4 发布为全局

管理员在设置面板每个页签（外观 / 壁纸）会看到「🔒 发布为全局」按钮：

1. 点击后把当前选择写入 `localStorage("leleo_global_config")`，自己刷新也保留；
2. 弹窗给出一段可直接粘贴进 `src/config.js` 的 JSON，复制  覆盖  重新构建部署即对所有访客生效。

### 3.5 实现文件

- `src/app.js`：`checkAdminToken()` / `isAdminNow()` / `isAdmin` data 字段 / `setMainProperty()` cookie 判断 / localStorage global_config 合并
- `src/App.vue`：设置齿轮按钮 + 设置面板整体 `v-if="isAdmin"`
- `src/components/tabs/tab1.vue` / `tab2.vue`：🔒 发布为全局按钮
